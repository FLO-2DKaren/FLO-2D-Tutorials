Import the FLO-2D *.DAT files into the ProjectÂ¶

To import a FLO-2D data files into a Geopackage:

  1. Click the Import GDS Files

  1. Navigate to the FLO-2D *.DAT files

  2. Select CONT.DAT and click Open.

  1. Select the desired components and click OK

  1. The message bar will tell state the files were imported.

  2. The dialog box will show the files that were imported.
