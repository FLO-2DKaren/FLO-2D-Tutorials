Create a raster¶

Overview¶

There are many uses for rasterized data in QGIS. It loads faster and is easier to style. FLO-2D spatial data and results can quickly be rasterized with this process.

Rasterize Grid Layer¶

  1. In this example, the grid elevation raster is used. To create a grid elevation raster, use the Saga-Rasterize tool.

  2. On the Main QGIS Menu, click Processing>Toolbox.

  1. Enter the search term Rasterize in the Processing Toolbox search feature. Double click the Saga Rasterized tool.

  1. Change the dialog box as shown below and click Run.

  2. Cell size is the grid element size.

  3. Fit = cells will ensure that the raster pixels are centered on the grid elements.

  4. Saga-Rasterize is a good raster generator for FLO-2D data because it builds a raster with hidden null data.

  5. If you want a temporary layer, leave the Rasterized field emptly.

  6. If you want a permanent file, use the Rasterize/Browser button to place the file in a directory. In this example a temporary file is used.

  1. Shapes = Grid Layer

  2. Attribute = elevation or whatever attribute you want to rasterize.

  3. Method for Multiple is default unless you want something in a different resolution from the source data.

  4. Method for lines = 0. Processing won’t run without this value. It isn’t important.

  5. Method for lines = 0. Processing won’t run without this value. It isn’t important.

  6. Target = float

  7. Output extent is important. Choose the Grid Layer

  1. Fit = cells

  2. Rasterized = Temporary or browse to a file locaiton.

Save the Permanent layer¶

  1. In this example, the raster that is created is a temporary layer called Rasterize.

  2. This layer can be saved to a permanent layer using the QGIS Save As feature. T

  3. To save a temporary layer, right click the layer and click Export/Save As.

  1. Navigate to the location and save the file.

  2. Fill out the Save File data and click OK Most of the data is default for this type of processing.
